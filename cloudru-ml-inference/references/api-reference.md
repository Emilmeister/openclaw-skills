# Cloud.ru ML Inference — API Reference

## Overview

The `inference-clients` Python package (import as `inference_activity`) provides two main clients:

1. **ModelRunUsersApiClient** — BFF client for CRUD operations on model runs (managing deployments)
2. **CallModelApi** — client for invoking deployed models (inference requests)

Base URL for BFF: `https://console.cloud.ru`
Base URL for inference calls: `modelrun.inference.cloud.ru` (each model run gets `https://<model_run_id>.modelrun.inference.cloud.ru`)

---

## Client Setup

```python
from uuid import UUID
from evoapp_http.auth import IAM_Auth
from http_client_retries import with_retries_failed_requests, with_retries_if
from inference_activity.clients import IAM_HttpClient, ModelRunUsersApiClient, options
from inference_activity.clients.inference.call_model_run_client import CallModelApi

# IAM client for token management
iam_client = IAM_HttpClient("https://iam.api.cloud.ru/")

# Auth handler (auto-obtains and refreshes bearer tokens)
auth = IAM_Auth(
    key_id=CP_CONSOLE_KEY_ID,
    key_secret=CP_CONSOLE_SECRET,
    iam_client=iam_client,
)

# BFF client for managing model runs
bff_client = ModelRunUsersApiClient(
    "https://console.cloud.ru",
    options.with_auth(auth),
    options.with_autogenerated_request_id(),
    options.with_autogenerated_traceparent(),
    with_retries_failed_requests(),
)

# Inference call client
call_client = CallModelApi(
    "modelrun.inference.cloud.ru",
    options.with_auth(auth),  # add if model run has auth enabled
    options.with_autogenerated_request_id(),
    options.with_autogenerated_traceparent(),
    with_retries_failed_requests(),
)
```

## BFF API Methods (ModelRunUsersApiClientV1)

All accessed via `bff_client.v1.<method>()`. All accept optional `anonymous: bool` and `auth: AuthTypes`.

| Method | HTTP | Description |
|--------|------|-------------|
| `get_list_model_run(project_id, params={"limit": N, "offset": M})` | GET | List model runs (paginated) |
| `get_details_model_run(project_id, model_run_id)` | GET | Get single model run |
| `create_model_run(project_id, model_run_body_data)` | POST | Create model run |
| `update_model_run(project_id, model_run_id, model_run_body_data)` | PUT | Update model run |
| `delete_model_run(project_id, model_run_id)` | DELETE | Delete model run |
| `suspend_model_run(project_id, model_run_id)` | PATCH | Suspend running model |
| `resume_model_run(project_id, model_run_id)` | PATCH | Resume suspended model |
| `get_history_model_run(project_id, model_run_id)` | GET | Get event history |
| `get_quota_usage(project_id)` | GET | Get GPU/CPU quota usage |
| `get_framework_versions(project_id, params)` | GET | List runtime templates |

## Inference Call Methods (CallModelApiV1)

All accessed via `call_client.v1.<method>()`.

| Method | Endpoint Pattern | Use Case |
|--------|-----------------|----------|
| `call_model_run_with_compatible_openai(model_run_id, response_schema, model_run_request_body_data)` | `/v1/chat/completions` | OpenAI-compatible text generation (Ollama, vLLM, SGLang) |
| `call_model_run_transformers_diffusers(model_run_id, model_run_name, response_schema, model_run_request_body_data)` | `/v2/models/{name}/infer` | Transformers/Diffusers inference |
| `call_embedding_model_run_with_compatible_openai(model_run_id, response_schema, model_run_request_body_data)` | `/v1/embeddings` | Embeddings |
| `call_rerank_model_run_with_compatible_openai(model_run_id, response_schema, model_run_request_body_data)` | `/v1/rerank` | Reranking |
| `ping_model_run_with_compatible_openai(model_run_id)` | `/v1/models` | Health check (OpenAI-compat) |
| `ping_model_run_transformers_diffusers(model_run_id, model_run_name)` | `/v2/models/{name}/ready` | Health check (Transformers/Diffusers) |

## Data Models

### ModelRunCreateItem (create/update request)

```python
from inference_activity.models.inference.bff.v1.requests import (
    ModelRunCreateItem,
    ModelSourceHuggingFace, ModelSourceOllama, ModelRegistry, ModelScope,
    Repository, Scaling, KeepAliveDuration,
    ScalingRulesConcurrencyType, ConcurrencyType,
    VllmOptions, VllmOptionsDynamic, DynamicOptions, DynamicOption,
    OllamaOptions, TransformersOptions, DiffusersOptions,
    Options, Auth,
)
from inference_activity.enums import (
    FrameworkTypeEnum, ResourcesTypeEnum, ModelTaskType,
    AuthTypeEnum, ParameterType,
)
```

Key fields:
- `name` — display name
- `framework_type` — FrameworkTypeEnum
- `resource_type` — ResourcesTypeEnum
- `model_task_type` — ModelTaskType
- `model_source` — one of: ModelSourceHuggingFace, ModelSourceOllama, ModelRegistry, ModelScope
- `serving_options` — framework-specific options (VllmOptions, OllamaOptions, etc.)
- `scaling` — autoscaling config (min_scale, max_scale, scaling_rules, keep_alive_duration)
- `gpu_count` — number of GPUs
- `gpu_gb_memory` — GPU memory in GB
- `runtime_template_id` — specific runtime version (get from `get_framework_versions`)
- `options` — auth and logging options

### ModelRun (response entity)

Key fields: `model_run_id`, `project_id`, `name`, `status`, `framework_type`, `resource_type`, `model_source`, `serving_options`, `scaling`, `gpu_count`, `gpu_gb_memory`, `public_url`, `created_at`, `updated_at`

### Model Sources

- **HuggingFace**: `ModelSourceHuggingFace(hugging_face_repository=Repository(repo="org/model", revision="main"))`
- **Ollama**: `ModelSourceOllama(ollama=Repository(model="tinyllama:latest"))`
- **Model Registry**: `ModelRegistry(model_registry=Repository(repo="registry-path", model="model-name"))`
- **ModelScope**: `ModelScope(model_scope=Repository(repo="org/model"))`

### Inference Request/Response Models

**OpenAI-compatible (Ollama)**:
- Request: `ModelTextToTextGenerationInputsRequestsOllama(model, messages, keep_alive)`
- Response: `ModelTextToTextGenerationResponseOllama` (choices, usage)

**OpenAI-compatible (vLLM/SGLang)**:
- Request: `ModelGenerateInputRequestVllmSglang(model, messages, temperature, top_p, ...)`
- Response: `ModelGenerateResponseVllm` / `ModelResponseGenerateSglang`

**Embeddings**:
- Request: `ModelEmbeddingInputRequestVllm(model, input=["text1", "text2"])`
- Response: `ModelEmbeddingResponseVllm` (data with embedding vectors)

**Rerank**:
- Request: `ModelRerankInputRequestVllm(model, query, documents)`
- Response: `ModelRerankResponseVllm` (results with relevance_score)

**Transformers**:
- Request: `ModelTextToTextGenerationInputsRequestsTransformers(inputs=[ModelGenerationInput(...)])`
- Response: `ModelGenerationResponseTransformers`

**Diffusers (image gen)**:
- Request: `ModelTextToImageRequestDiffusers(inputs=[ModelGenerationInput(...)])`
- Response: `ModelTextToImageResponseDiffusers`
